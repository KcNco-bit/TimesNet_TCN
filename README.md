# TimesNet_TCN
TimesNet-TCN is a model for industrial soft sensing, with its TimesNet structure based on the paper https://arxiv.org/abs/2210.02186 proposed by Wu et al. The model testing architecture and comparative model are based on the Time Series Library: https://github.com/thuml/Time-Series-Library
Configure the running parameters in the "run.ipynb" file and run it.
